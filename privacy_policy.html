Budgee – Privacy Policy

Last updated: February 2025

Developer: Roi Dital

This Privacy Policy explains how the Budgee app (“the App”) collects, uses, and protects your information. By using Budgee, you agree to the practices described here.

1. Information We Collect

1.1 Notification Access (Google Wallet / Google Pay)

Budgee requests permission to read push notifications. This permission is used exclusively to read transaction notifications from Google Pay / Google Wallet.

From these notifications, the App extracts only:

  • Merchant name
  • Transaction amount

No other notifications from any other apps or sources are read, stored, parsed, or collected.

1.2 SMS Access (Israeli Credit Card Providers Only)

Budgee may request permission to read SMS messages. The App reads SMS content only from known Israeli credit card issuers, including:

  • Visa CAL
  • Isracard
  • Diners
  • Other established Israeli financial institutions

The App extracts only:

  • Merchant name
  • Transaction amount
  • Installments count (if included)
  • Last 4 digits of the credit card (used solely for duplicate prevention)

The App does not read or process SMS messages from any other sender or source.

1.3 Financial Data You Enter

Your financial data may include:

  • Expenses and income
  • Budget settings
  • Loans
  • Salary (optional, only if provided)

This data is stored locally on the device in an encrypted SharedPreferences storage, and is also synced securely (TLS) to Firebase Realtime Database under your unique user ID.

1.4 Authentication Information

Budgee uses Firebase Authentication to manage user login. Firebase securely stores your:

  • Email address
  • Password (hashed/salted using Firebase security protocols)

This information is not stored or handled by the App directly.

2. How We Use Your Information

The information collected is used solely for the core functionality of the App:

  • Automatically recording expenses from Google Wallet notifications
  • Automatically recording expenses from financial SMS messages
  • Displaying and syncing your budget and financial data

Your data is never sold, shared, or used for advertising.

3. Optional Data Sharing With OpenAI (ChatGPT)

Some optional features in Budgee allow users to receive insights or summaries using OpenAI's ChatGPT API.

Important:

  • You are shown an in-app privacy warning before any data is sent.
  • The exact information being shared is clearly listed.
  • You must explicitly approve the sharing before the feature is used.
  • If you decline, no financial data is shared and the feature remains disabled.

Data is only shared with OpenAI when the user directly authorizes it.

4. Data Security

We take security seriously:

  • Local storage uses encrypted SharedPreferences
  • All network communication uses TLS encryption
  • Firebase Realtime Database is protected with strict security rules
  • User data is isolated per account and not visible to other users

5. Data Retention & Deletion

Your financial data is stored for as long as your account remains active.

You may request deletion of your account and all associated data at any time.

To request deletion, please visit:
Account Deletion Request Page

Once deletion is requested, all data in Firebase Authentication and Firebase Realtime Database associated with your account will be permanently removed.

6. Children’s Privacy

Budgee is not intended for use by individuals under the age of 13. We do not knowingly collect information from children.

7. Changes to This Policy

The Privacy Policy may be updated periodically. Updates will be posted on this page with a new “Last updated” date.
